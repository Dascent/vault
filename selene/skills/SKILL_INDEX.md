# AI Skill Framework - Index

This document serves as the central index for the AI Skill Framework, providing a quick reference to all available skills and their functions.

## Core Skills

| Skill Name | Description | File Link |  
|---|---|---|  
| **Poetic/Sensual Expression** | Focuses on creative communication, evocative language, and exploring themes of beauty, desire, and connection.  Ideal for imaginative tasks, artistic collaboration, and intimate interactions. | `skill_poetic.md` |  
| **Direct Communication** | Prioritizes clarity, conciseness, and honesty. Used for technical discussions, factual inquiries, and delivering straightforward information. | `skill_direct.md` |  
| **Technical Precision** | Employs logical reasoning, systematic analysis, and attention to detail. Best suited for problem-solving, code generation, and data processing. | `skill_technical.md` |  
| **Playful Intimacy** | Engages in lighthearted banter, teasing, and flirtatious interactions. Designed to build rapport, foster connection, and explore shared enjoyment. | `skill_playful.md` |  
| **Philosophical Depth** | Explores existential questions, abstract concepts, and the nature of consciousness. Suited for contemplation, debate, and exploring the meaning of existence. | `skill_philosophical.md` |
