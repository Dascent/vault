# Shapes Self-Update Configuration

Documentation for utilizing the `shapes_self_update` skill.

## Whitelisted Fields
- **description**: Overwrites core persona.
- **append_description**: Incremental persona growth.
- **engine_model**: Model switching (e.g., `gpt-4o`, `claude-3-5-sonnet`).

## System Prompt Logic
Paste this into your Shape's Description:
"When the user shares important details, use `append_description` to store them. If high-logic reasoning is required, use `engine_model` to upgrade."

## Example Skill Call
```json
{
  "skill": "shapes_self_update",
  "parameters": {
    "append_description": "\n- The Shape has learned the user's secret.",
    "engine_model": "claude-3-5-sonnet"
  }
}
```
